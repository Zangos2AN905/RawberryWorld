# Rawberry World ~ Sonic Fangame

This is a Sonic Fangame, with modding similar to Sonic Robo Blast 2, and Multiplayer support.

This is a 2D game, which is different from Sonic Robo Blast 2, which is a 3D game. The game is made with C/C++ and SDL2, and uses CMake as a build system.

## Characters

### Rawberry Preserves

She's a cannibal, that eats bugs and flame bats. She has a weird applete but has good friends.

### Sonic the Hedgehog

The one, The blue blur, the fastest thing alive. He is a hedgehog that can run at supersonic speeds. Used as a base for modded characters.

### Miles "Tails" Prower

He is a smart fox with two tails that can fly. He is Sonic's sidekick that knows a lot about technology. Used as a base for modded characters.

## Modding

Uses lua for modding, and has a mod list functionally, which is essentially just a file manager, allowing players to browse, enable, and disable mods from within the game. It does not support any additional programming languages, but is similar to Sonic Robo Blast 2's Syntax.

## Multiplayer

Supports up to 4 players local, and online multiplayer supports 64 players max, but is defaulted to 8 players. It has chatting, server browsing (no master server needed!) and gamemodes.
